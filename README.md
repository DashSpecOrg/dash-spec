# dash-spec

Workspace scaffold for a dashboard specification parser and demo app.

## Structure

```text
.
|-- apps/
|   `-- demo/
`-- packages/
    `-- dash-spec/
```

## Packages

- `packages/dash-spec`: TypeScript library for parsing and validating dashboard specs.
- `apps/demo`: Next.js demo app that will render dashboards using the library.

## Status

This repository is currently scaffolded only. Core parsing, validation, and rendering behavior will be added next.
